Tomato Disease Classification

An advanced deep learning solution for early detection and classification of tomato plant diseases, achieving over 98% accuracy in identifying 9 distinct diseases and healthy plants.

Learn More

Research Overview

Project Significance

This research addresses the critical challenge of early disease detection in tomato plants, which is essential for sustainable agriculture and food security. Our deep learning-based solution provides rapid, accurate disease identification, helping farmers implement timely interventions and reduce crop losses.

Technical Methodology

Data Processing

  • ✓ Image preprocessing to 64x64 resolution
  • ✓ Advanced data augmentation techniques
    • - Horizontal and vertical flipping
    • - Random rotation and zooming
    • - Brightness variation
    • - Contrast adjustment

Model Architecture

  • ✓ Convolutional Neural Network (CNN)
  • ✓ Key Components:
    • - Multiple convolutional layers
    • - Max pooling layers
    • - Dropout for regularization
    • - Dense layers for classification

Disease Categories

  • ✓ Bacterial Spot
  • ✓ Early Blight
  • ✓ Late Blight
  • ✓ Leaf Mold
  • ✓ Septoria Leaf Spot
  • ✓ Spider Mites
  • ✓ Target Spot
  • ✓ Yellow Leaf Curl Virus
  • ✓ Mosaic Virus
  • ✓ Healthy Plant Detection

Results and Impact

Performance Metrics

  • ✓ Overall Accuracy: 98.2%
  • ✓ Average Precision: 97.8%
  • ✓ Average Recall: 97.5%
  • ✓ F1-Score: 97.6%

Practical Applications

  • ✓ Real-time disease detection
  • ✓ Early intervention guidance
  • ✓ Reduced crop losses
  • ✓ Sustainable farming support
